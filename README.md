# Impostor Fútbol ⚽

**Impostor** es un juego de deducción social para jugar con amigos durante tus ratos libres en la universidad, el colegio o el trabajo. ¡Perfecto para grupos de 2 a 10 jugadores!

## 🎮 Juega ahora

👉 [**impostorfutbol-b3mvqqr6.manus.space**](https://impostorfutbol-b3mvqqr6.manus.space)

## 🧠 ¿Cómo se juega?

> **"Presiona las cartas para revelar futbolistas. Todos ven el mismo jugador, excepto el impostor que ve uno diferente. ¡Descúbrelo!"**

1.  **Elige tu categoría**: Selecciona el grupo de futbolistas con los que quieres jugar. (💡 *Cada categoría tiene diferentes futbolistas. ¡Pruébalas todas!*)
2.  **Número de jugadores**: Define cuántas personas participarán (de 2 a 10).
3.  **A jugar**: ¡A descubrir al impostor!

## ✨ Características

-   Multijugador (2-10 jugadores)
-   Múltiples categorías de futbolistas
-   Interfaz simple y rápida
-   Ideal para partidas cortas

## 🛠️ Tecnologías utilizadas

-   React
-   TypeScript
-   Tailwind CSS
-   Vite
